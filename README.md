💻 Projetos em C++

Este repositório contém uma coleção de programas desenvolvidos em C++, com foco em aprendizado, prática de lógica e desenvolvimento de soluções eficientes.

📚 Conteúdo

Os códigos incluem:

🧮 Algoritmos básicos
🔁 Estruturas de repetição (for, while, do-while)
🔀 Estruturas condicionais (if, else, switch)
📦 Funções
🧠 Vetores e matrizes
🧩 Programação orientada a objetos (POO)
🗂️ Manipulação de arquivos
⚙️ Uso da STL (Standard Template Library)

⚙️ Como compilar e executar
🔹 Usando g++
Compile o código:
g++ nome_do_arquivo.cpp -o programa
Execute:
./programa
🧰 Requisitos
Compilador C++ (g++ recomendado)
Terminal ou prompt de comando
